Help: Removing Administrators

Description:

This page allows the user to remove other users from the server.

Username Textbox:

This is where the user types in the name of the Admin that they want to remove from the server. Note, if an error is made in typing the name of an Admin, or the username entered does not exist, the request will fail. So take care when typing the name of the Admin.

Remove Admin Button:

Once the user has typed the name of the Admin they wish to remove from the server. Click the button to send the request to remove the Admin from the server.
