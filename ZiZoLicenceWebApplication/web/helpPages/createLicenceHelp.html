Help: Creating A Licence

Description:

This page is used to create a licence on the server. Simply fill in the required fields (Username, Server Address, Licence Length) To create the licence.

Username Textbox:

This is where the user types the name of the Licence they wish to create. While you can name the licence as you please, it is recommended that it is put in an email format (test@licence.co.uk), if you do not put the licence in this format, you will be prompted if you want to continue.

Server Address Textbox:

This is where the user types in the name of the server address, this field may not be required to be filled in as the system should automatically fill the textbox with your current URL.

Filter Textbox:

This is where the user types the applicable filters to the licence when the request is sent. Note, if a filter is spelled incorrectly, or does not exist, the filter will not apply to the newly created licecne. Remember to take care when typing your filters.

Licence Length Radio Button Set:

The user must select one of the three options available to them in the radio button set. The first two options have fixed times, however the third option must have a date manually input (the date automatically defaults to three days). If your browser does not include the calendar feature, the date must be input as 'dd-mm-yyyy'.

Create Licence Button:

Once all of the required fields have been filled, the user must click this button to send the request to the server.
