Help: Removing Licences

Description:

This page is for if the user wishes to remove a licence from another user (whether it is a permanent removal, or just a means in which to re issue a new licence). Note, if a licence has been lost/misplaced by the user, the regenerate licence option should be the first port of call, before trying to remove the lost licence.

Username Textbox:

This textbox is used to type the Licence key which you wish to remove. Note, if the name of the user is incorrectly spelled, or does not exist, no licence will be removed. Remember to take care when typing the name of the user.

Remove Licence Button:

Once the Licence Key textbox has been correctly inputted into. The user then clicks on the remove licence button, the webpage then sends a request to remove the licence from the selected user.

Search Textbox:

This textbox is used to search for specific Licences that match the pattern that is typed in. If the user does not have the 'check-licence' permission, the user will be unable to use this feature.
