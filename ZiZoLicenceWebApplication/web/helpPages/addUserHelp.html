Help: Adding Users

Description:

This page is used for the sole purpose of adding a user to the Licence Server. This user will be created without any permissions, and must be added seperately.

Username Textbox:

The Username textbox is where you must type the name of the new user that you are adding. It is highly recommended that the user being created is in the form of an email address (E.g. example@name.co.uk).

Password Textbox:

The Password textbox is where you must type the password for the user you are creating. Remember to set the password as something memorable, so as to not be locked out of the account due to forgetting the password.

Confirm Password Textbox:

This textbox is simply used to confirm you have indeed created the password you have intended to. The user must re type the password they have created in the original 'Password Textbox' to verify the password is correct. The form will not proceed until both the 'Password Textbox' and 'Confirm Password Textbox' match.

Add User Button:

Once all of the required fields are filled correctly. Click this button to send a request to create the new user.
