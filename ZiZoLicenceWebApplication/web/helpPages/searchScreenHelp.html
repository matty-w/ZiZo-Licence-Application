Help: The Licence Search Table

Description:

This page displays a list of the licences that meet the search criteria from the Remove Licence page, from here the User can view and delete multiple licences at once.

The Licence Search Table:

The Table shows three columns of information, the name of the licence (which is matched) against the pattern the user typed in on the previous page, the licence key for the corresponding licence and also the licecne length. Checkboxes are located to the left of each of the returned licences, this is where you select which licences you wish to delete.

The Return Button:

This button is used to return to the previous page (Which is the Remove Licence page). If you no longer wish to delete any licences from the returned search, simply click this button to return.

The Delete Button:

Once the user has selected which licences they wish to remove (by selecting them through the checkbox options to the left), the user clicks the delete button, a prompt will appear which will ask the user whether they are sure they wish to delete the licences that are selected, if the user clicks yes, the licences will be deleted, and the user will be returned to the Remove Licence page.
