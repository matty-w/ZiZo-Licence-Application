Help: Changing Password

Description:

This page is used for the sole purpose of changing the password for the logged in Administrator.

Username Textbox:

The Username Textbox contains the Username of the currently logged in User, this text is Uneditable. Unless the User has rights to change the passwords of any User.

Password Textbox:

This is where the User types the password they want their current password changed to, the password must be at least 8 characters long.

Confirm Password Textbox:

This textbox is simply used to confirm you have indeed created the password you have intended to. The user must re type the password they have created in the original 'Password Textbox' to verify the password is correct. The form will not proceed until both the 'Password Textbox' and 'Confirm Password Textbox' match.

Change Password Button:

Once all of the required fields are filled correctly. Click this button to send a request to create the new password.
