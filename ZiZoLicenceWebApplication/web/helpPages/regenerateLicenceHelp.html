Help: Regenerate A Licence

Description:

This page is used to recreate the Licence Key for a user that may have either accidentally deleted or moved the original key. This will avoid the issue of re issuing new licences to already licenced users.

Username Textbox:

The username textbox is used to input the name of the user you wish to regenerate a licence key for. Note, if the name of the user is incorrectly spelled, or does not exist, no licence will be re issued.

Regenerate Licence Button:

Once the name of the user that they with to regenerate the licence for has been inputted, this button should be clicked to send the request.
