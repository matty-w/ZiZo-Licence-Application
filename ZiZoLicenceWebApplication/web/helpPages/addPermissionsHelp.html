Help: Adding Permissions

Description:

This page is used to add permissions (allowances) for the selected user. This can include allowances such as the ability to view Administrator Logs to the ability to change the passwords of users.

Username Textbox:

The Username textbox is where you must enter the name of the user you are adding a permission for. Note, if there is an error when typing the name, or the user does not exist on the server, the permission will not be created. So take care when typing the name.

Permissions Drop Down Box:

This drop down box is what is used to select the permission you wish to attatch to the user. You can add as many permissions as you want to a specified user, but it must be done one at a time.

Add Permissions Button:

Once the user and permission has been selected, click the button to send a request to the server to add the permission for the user.
