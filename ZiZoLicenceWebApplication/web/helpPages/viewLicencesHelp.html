Help: Viewing Licences

Description:

This page is used to view the various logs made by the users registered to this server. All thier actions will be recorded for viewing in the table below.

Filter For Admin Checkbox:

The user checks this checkbox if they are only requiring the licences of Administrators. All other licences will not be shown.

Filter Textbox:

The user types the filters they wish to apply when searching through the licence history. Note, if a filter is incorrectly spelled, or does not exist, the filter will not apply to the search. Remember to take care when typing your filters.

View Logs Button:

Once the user has selected the filter they with to apply (and chosen whether or not to only search for Admin licences), the user clicks this button. The table will then display the search results.

Export Logs Button:

This button is used to export the table of results to a 'CSV' file. This will take whatever is currently in the table of results and save them to a file on your computer. Remember, if the table is empty, the button will still save an empty 'CSV' file if the button is clicked.
